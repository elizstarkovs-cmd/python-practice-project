Для анализа использовался файл news_data.csv, содержащий приблизительно 100 записей. Структура данных:
id — уникальный идентификатор;
title — заголовок;
category — категория (например, "Политика", "Экономика", "Спорт");
views — число просмотров (целое);
publication_date — дата публикации (формат YYYY-MM-DD).
Этапы работы:
Подготовка окружения:
Создана рабочая директория проекта с файлами news_analyzer.py (основной скрипт) и news_data.csv (исходные данные). Для выполнения использована среда Python 3.13.9
Разработка скрипта news_analyzer.py:
Реализованы:
Обработка аргументов командной строки для указания пути к CSV-файлу.
Чтение данных через модуль csv.
Хранение данных в виде списка словарей.
Агрегация статистики по категориям с использованием словаря.
Функциональность анализа:
Скрипт выполняет:
Подсчет общего числа новостей.
Определение самой популярной новости (максимум просмотров).
Расчет распределения новостей по категориям с сортировкой по убыванию.
Вычисление среднего числа просмотров на новость.
Формирование отчета:
Результаты выводятся в консоль в структурированном виде и включают:
Общее количество новостей.
Данные о самой популярной новости.
Отсортированную таблицу по категориям.
Среднее значение просмотров.
Тестирование:
Проверена корректность работы скрипта на файле news_data.csv. Убедились в точности расчетов, обработке строк и чисел, а также в корректной сортировке и выводе статистики, включая повторяющиеся категории.
Особенности реализации:
Использование CSV-формата обеспечивает универсальность и совместимость с различными источниками данных.
Консольный интерфейс позволяет легко интегрировать скрипт в автоматизированные процессы.
Применение стандартных структур данных Python (списки, словари) обеспечивает эффективность обработки и простоту дальнейшего расширения функциональности.
Результат:
Задание выполнено в полном объеме. Скрипт news_analyzer.py успешно анализирует CSV-файл с новостями, формирует статистический отчет и демонстрирует владение ключевыми аспектами обработки данных в Python.
Файлы проекта:
